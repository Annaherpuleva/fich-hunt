# UI Terminology Checklist

Use this checklist for all user-facing texts in `client/src`.

## Forbidden terms in UI copy
- `smart contract`
- `programId`
- `on-chain` / `onchain`
- `смарт-контракт`
- `контракт` (when referring to core game processing)
- `ончейн`

## Preferred alternatives
- `game service`
- `backend`
- `server-side processing`
- `database`
- `system operation`
- `игровой сервис`
- `серверная обработка`
- `база данных`
- `операция в системе`

## Quick review steps
1. Scan changed files for forbidden terms before commit.
2. Verify Terms / Privacy / Contacts wording stays aligned with server-driven model.
3. Keep internal technical names (code-only identifiers) out of user-facing UI strings.
